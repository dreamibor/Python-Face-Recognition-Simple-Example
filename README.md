Python Face Recognition Simple Example
Python简单实践系列（一）Python照片人脸检测

功能：
1、读取照片，显示在GUI中
2、识别人脸，并画出人脸、眼睛的位置
3、显示人脸信息，包括年龄、性别、人种

环境：
Ubuntu 14.04 LTS + Python 2.7.10 + Face++ 人脸识别API

模块：
1、Pillow做图片处理
2、PyQt4（Ubuntu自带）作为GUI库
3、sys、time等内置模块。

说明：
1、我的代码全在rec.py文件中
2、如果有.idea隐藏文件夹，为Face++ API默认生成
3、facepp.py文件为Face++ SDK提供，请搜索下载
4、pic文件夹下为测试图片
